Ender-2-Pro-BTT Upgrade to BTT SKR Mini E3 v2 + TFT35 E3 v3

The version of Marlin [Marlin-bugfix-2.1.x.zip] is already configured for the Ender 2 Pro-BTT upgrade and is included in the code zip file

that can be downloaded from the THe-300lb-Gorilla/Ender-2-Pro-BTT/Marlin Firmware Source Code.

Just unzip file, and open in VS Code, make changes as you see fit if needed, and compile firmware. 

(If you make incorrect changes, the firmware may not compile, MAKE CHANGES AT YOUR OWN RISK!)

Or, you can use the Pre-Compiled firmware in the main branch, just remember to change the name of the file as instructed in the line below.

Change firmware file name from "firmware-update-6-12-2022.bin" to: firmware.bin before loading it onto the Ender 2 Pro

Upgrade the Ender 2 Pro To A BTT-SKR-Mini-E3-v2 + TFT35-E3-v3 Touch Screen Display / Classic Marlin Dislay

This upgrade requires not only the BTT SKR mini E3 v2 and the TFT35 E3 v3

This upgrade requires a BTT SKR Mini E3 v2 Main Board and TFT35 E3 v3 bundle. Available on Amazon.com

This upgrade also requires NEW Motor Cables. 6 pin to 4 pin, Ender 3 motor cables. Available on Amazon.com

This upgrade also requires a longer 10 pin to 10 pin LCD ribbon cable. Available on Amazon.com

You will need to 3D Print a control box for the TFT35 Touch Screen. There is a Zip file that contains these files.
